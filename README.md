# Jiban Niraula Portfolio

Production-ready static portfolio built with plain HTML, CSS, and JavaScript.

## Run locally

```bash
python -m http.server 5173
```

Open `http://localhost:5173`.

## Publish

Upload the contents of this folder to your hosting root / `public_html`.

## Image

Replace `assets/jiban.png` with the real profile image using the same filename.
No overlay cards are placed on top of the profile image.
